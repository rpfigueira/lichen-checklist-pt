# Checklist of Lichens and Lichenicolous Fungi from Mainland Portugal

### Data analysis of manuscript 

This repository contains scripts build to perform data analysis and graphical 
outputs of the manuscript *Checklist of Lichens and Lichenicolous Fungi from Mainland Portugal*. 

It includes the following scripts:

| script name | description | used in |
|-------------|-------------|-------------|
| tables_table1 | prepares data for summary table | Table 1 |
| charts_fig1_sunburst_taxa | creates a sunburst plot for taxa tree included in the checklist | Figure 1 |
| charts_fig2_province_no_labels | performs exploratory data analysis and creates plots per province with the number of occurrences per decade, and accumulated number of taxa through time  | Figure 2 |
| compose_fig2_map_province | creates the composition of charts and map | Figure 2 |
| charts_fig4_substrate | creates the radial plots of the number of species per substrate type | Figure 4 |
